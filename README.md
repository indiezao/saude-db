# saude-db
Uma aplicação para emular um banco de dados local de um sistema de saúde.

> ### ⚠️ Clínica ainda em Construção
> Esta aplicação ainda não está finalizada. Nem todas as funcionalidades estão completas e você pode encontrar erros na execução do programa.

*Painel de listagem de Pacientes*

### Funções:
- Gerenciamento de Pacientes e Médicos
- Agendamento de Consultas
- Compatibilidade multi-plataforma (Com python3)


### Bibliotecas:
- tabulate: para criar as tabelas de Pacientes, Medicos e Agendamentos
- datetime: obter a data e hora e gerenciar os Agendamentos

Essa aplicação **NÃO** deve ser utilizada em ambiente de trabalho real.
